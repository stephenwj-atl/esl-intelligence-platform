MODULE 2 — GRANT TRANSLATION (NEW — HIGH PRIORITY)
Create:
GrantOutput {
  impactDeliveryProbability
  impactEfficiencyScore
  costPerOutcome
  disbursementRisk
  utilisationRate
}
LOGIC
A. Impact Delivery Probability
base = 100

subtract:
- environmental risk penalty
- data confidence penalty
- sector complexity penalty
B. Impact Efficiency Score
efficiency = f(risk, confidence, environmental volatility)
C. Cost Per Outcome
adjust base cost by:
- environmental constraints
- rework probability
- delay likelihood
D. Disbursement Risk
if confidence < threshold OR high risk:
  disbursementRisk = HIGH
E. Utilisation Rate
utilisation = projected % of grant actually used effectively
MODULE 3 — BLENDED TRANSLATION (NEW — CRITICAL)
Create:
BlendedOutput {
  grantPercentage
  loanPercentage
  concessionalityLevel
  transitionTimeline
  viabilityThreshold
  firstLossEstimate
  crowdInRatio
}
LOGIC
A. Grant / Loan Split
grant% = f(risk, confidence, validation gaps)
loan% = remainder
B. Transition Timeline
timeline = sequence of:
- validation
- monitoring
- intervention
→ until risk < threshold
C. Viability Threshold
loanEligible = risk < X AND confidence > Y
D. First Loss Estimate
firstLoss = % of capital required to absorb risk
E. Crowding-In Ratio
privateCapital = f(risk reduction via grant)
